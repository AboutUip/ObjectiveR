# ObjectiveR (Obr)

ObjectiveR（简称 **Obr**）是一门解释型、强类型编程语言，源文件扩展名为 `.obr`。  
它融合了面向对象、过程式、函数式以及泛型编程等多种编程范式，目标是在统一语义下提供清晰、可扩展且具备工程实践能力的语言体验。

## 语言简介

- **语言名称**：ObjectiveR  
- **简称**：Obr  
- **文件扩展名**：`.obr`  
- **执行方式**：解释执行（Interpreted）  
- **实现语言**：Java（原生支持跨平台运行）  
- **类型系统**：强类型（Strongly Typed）  
- **编程范式**：面向对象、过程式、函数式、泛型编程  
- **语法规则**：严格大小写检查（Case-Sensitive）

## 设计目标

ObjectiveR 致力于在多范式编程能力之间取得平衡：

- 使用面向对象能力组织复杂系统；
- 使用过程式风格编写直观、可控的逻辑流程；
- 使用函数式风格提升表达力与可组合性；
- 使用泛型能力提升复用性与类型安全。

## 版本兼容性说明

ObjectiveR 的各个版本 **互不兼容**。  
在升级或迁移版本时，请以对应版本文档和工具链为准，避免直接混用不同版本的语法与运行时行为。

## 语法约定

ObjectiveR 采用严格大小写检查。  
标识符在不同大小写下会被视为不同符号（例如 `Name` 与 `name` 不等价），请在命名与调用时保持一致。
