# ObjectiveR 语言规范索引

本目录用于按主题拆分维护 ObjectiveR（Obr）语言规范，降低单文档维护成本。

## 规范索引

- `overview.md`：规范总览（范围、版本策略、基础约定）
- `preprocessor.md`：预处理指令规范（`#VERSION`、`#LINK`）
- `runtime.md`：执行模型规范（启动输入、唯一入口、根目录判定、资源扫描与执行入口）
- `visibility.md`：可见性规范（`public`、`private`、默认访问级别）
- `static.md`：`static` 关键字规范（静态可访问、不可逆与唯一性）
- `termination.md`：语句终止与换行规范（分号结束、预处理按行结束、逗号续写）
- `moduleR.md`：`.mr`（moduleR）文件规范（函数头/宏定义、扫描、`import` 导入与资源标识）
- `comments.md`：注释规范（`//` 与 `/* ... */`）
- `types.md`：基础数据类型规范
- `scope.md`：作用域与标识符解析规范
- `operators.md`：运算符与隐性转化规范

## 维护原则

- 新增语言能力时，优先新增或更新对应专题文档。
- 避免将所有规则堆叠到单一文件中。
- 若规则存在版本差异，需在对应专题文档中显式标注。
- 所有专题文档均为技术规范文档，使用“必须/应当/可选”类规范措辞，避免实现导向描述。
