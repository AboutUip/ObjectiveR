# ObjectiveR 可见性与访问修饰规范

## 1. 关键字

ObjectiveR 定义以下可见性关键字：

- `public`
- `private`

以上关键字大小写敏感。

## 2. 作用位置

- `public` 与 `private` 可附加在部分表达式或声明语句之前。
- 具体可附加的语法对象清单在后续语法规范中补充。
- 在清单补充前，规范仅定义其访问语义与默认行为。

## 3. 默认规则

- 当目标语法对象未显式标注可见性关键字时，默认可见性为 `public`。

## 4. `public` 语义

- `public` 用于显式声明目标对象可被访问。
- 在语言空间访问规则允许的前提下，其他 `.obr` 文件可访问 `public` 目标对象。

## 5. `private` 语义

- `private` 用于声明目标对象仅允许在当前 `.obr` 文件内访问。
- 即使当前文件处于开放语言空间内，`private` 目标对象仍不得被其他 `.obr` 文件访问。
- 即使目标对象具有静态属性（static），`private` 限制仍然生效，不得跨文件访问。

## 6. 优先级与判定关系

- 可见性判定与语言空间判定必须同时满足才允许访问。
- `private` 的文件内限制优先于开放语言空间。
- 因此，语言空间开放并不意味着可绕过 `private`。
