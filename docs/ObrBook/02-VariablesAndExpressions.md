## 2.1 `var[type]`：声明并（可选）初始化

最常见的声明长这样：

```text
var[int] x = 10;
```

也可以一次声明多个变量，用逗号分隔，并以分号结束：

```text
var[int] a = 1,
         b = 2;
```

### 2.1.1 初值规则：一条 `var` 声明要么全带要么全不带

下面这种“混用”写法是不允许的：

```text
// 错误示意：同一条 var 声明里有的带初值、有的没带
var[int] a = 1, b;
```

正确写法是二选一：

```text
// 全带初值
var[int] a = 1, b = 2;

// 全不带初值（默认值由类型决定）
var[int] a, b;
```

## 2.2 赋值：`=` 与复合赋值

### 2.2.1 普通赋值

```text
x = x + 1;
```

### 2.2.2 复合赋值

```text
x += 3;   // 等价于 x = x + 3;
x -= 3;   // 等价于 x = x - 3;
```

复合赋值可以理解为：“把右边运算的结果加回到左边变量里”。

## 2.3 自增自减：`++` / `--`

你已经在 `demo/main.obr` 里见过它：

```text
min--;
min++;
```

更完整地说（你先当作语句使用）：

```text
x++;   // 后缀形式：先用旧值，最后再更新（本章不必深入）
x--;   // 后缀形式
--x;   // 前缀形式
+x;   // 前缀形式
```

注意：自增自减只能对数值类型用（比如 `int`）。

## 2.4 表达式：你至少需要会用这些

### 2.4.1 数值运算

```text
var[int] a = 2, b = 3;
var[int] c = a + b;
var[int] d = a * b;
var[int] e = a % b;  // 取模
```

### 2.4.2 相等与比较（用于条件判断）

这些运算会产生 `boolean`：

- `==`、`!=`
- `<`、`<=`、`>`、`>=`

示例：

```text
var[boolean] ok = (a == b);
```

（在下一章的 `if/while` 里你更常直接把它们写进条件括号里。）

### 2.4.3 `+` 作为“字符串拼接”

当 `+` 的一侧是 `string` 或 `char` 时，`+` 表示字符串拼接（不是数值相加）。

示例：

```text
var[int] x = 7;
std::rout("x=" + x); // 把数字转成文本，再拼接
```

## 2.5 一个完整的小练习（建议你直接复制跑一遍）

```text
#VERSION 1
#LINK /

deRfun main():void{
    var[int] x = 2,
             y = 3;

    x += 10;
    y--;
    std::rout("x=" + x);
    std::rout("y=" + y);
}
```

## 2.6 常见错误清单（本章只列最容易踩的）

- `var[int] a = 1, b;`：同一条 `var` 声明里混合了“带初值”和“不带初值”
- 忘记 `;`：例如 `std::rout("hi")` 少了分号
- 把字符串 `+` 当成数值加法：`"1" + 2` 不会做数字运算
