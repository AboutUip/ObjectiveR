# ObrBook：ObjectiveR 入门手册（面向严格零基础）

这份手册用于帮助你从“完全不知道 Obr”到“能写出可运行的最小程序，并逐步学会模块、控制流与 #LINK”。

阅读建议（按顺序）：

1. [00-HowToRead.md](00-HowToRead.md)：学习方式与约定
2. [01-StartFromMainObr.md](01-StartFromMainObr.md)：从 `main.obr` 开始写出第一个程序
3. [02-VariablesAndExpressions.md](02-VariablesAndExpressions.md)：`var`、赋值、自增自减、常见表达式
4. [03-IfWhileAndTernary.md](03-IfWhileAndTernary.md)：条件分支、循环、三元 `?:`
5. [04-ModulesMRAndFunctions.md](04-ModulesMRAndFunctions.md)：为什么需要 `.mr`，以及如何声明/实现函数
6. [05-VERSION-LINK-AndProjectSpace.md](05-VERSION-LINK-AndProjectSpace.md)：`#VERSION` 与 `#LINK`（语言空间）
7. [06-Exercises.md](06-Exercises.md)：配套练习题与自测清单

你不需要一次看完；可以每学一章就自己把示例代码跑起来，对照理解。
