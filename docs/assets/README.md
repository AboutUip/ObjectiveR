# `docs/assets/`

**文档与 README 中引用的是 PNG**（便于 GitHub / 编辑器预览）。**源稿为 SVG**，由维护者自行导出 PNG，文件名与下表对应。

| PNG（引用名） | 对应 SVG 源 |
|:--|:--|
| `obr-logo.png` | `obr-logo.svg` |
| `obr-logo-wide.png` | `obr-logo-wide.svg` |
| `mit-badge.png` | `mit-badge.svg` |
| `ornament-legal.png` | `ornament-legal.svg`（若存在） |
| `obr-mark.png` | `obr-mark.svg`（若存在） |

Markdown 示例：

```markdown
![Obr](assets/obr-logo.png)
```
