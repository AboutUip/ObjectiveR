# `docs/assets/`

仓库内徽标与装饰图为 **SVG**；根目录 `README.md` 与 `docs/` 下 Markdown 中的 `<img>` 直接引用这些文件（路径按所在文件调整）。

| 文件 | 用途 |
|:--|:--|
| [`obr-logo.svg`](obr-logo.svg) | Obr 方形图标（128×128，无外链字体） |
| [`obr-logo-wide.svg`](obr-logo-wide.svg) | 横版：图标 + 抽象条带（无字体依赖） |
| [`mit-badge.svg`](mit-badge.svg) | MIT 协议装饰条（文档点缀） |
| [`ornament-legal.svg`](ornament-legal.svg) | 法律页顶栏装饰 |
| [`obr-mark.svg`](obr-mark.svg) | 法律页小标记 |

引用示例：

```markdown
![Obr](assets/obr-logo.svg)
```
